# README — images/

Drop your photo here as `jake.jpg` (or `jake.png`, `jake.webp`) and the site will automatically use it in the hero section.

The image is displayed as a circle, ~208×208px. Best results: a square crop, face/shoulders. Any standard photo works — the CSS handles the circle mask.

If the file isn't present, the site shows a placeholder silhouette.

Other images (project screenshots, etc.) can live here too.
